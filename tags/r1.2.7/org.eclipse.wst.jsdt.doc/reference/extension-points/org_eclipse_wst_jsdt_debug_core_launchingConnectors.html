Launching Connectors

Identifier:
org.eclipse.wst.jsdt.debug.core.launchingConnectors

Since:
1.0

Description:
This extension point allows contributors to specify their own definition of a connector

Configuration Markup:

<!ELEMENT extension (connector)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED

>


<!ELEMENT connector EMPTY>

<!ATTLIST connector

class CDATA #REQUIRED

id    CDATA #REQUIRED

>

  • class - The connector class that implements org.eclipse.e4.languages.javascript.jsdi.connect.Connector
  • id - An id unique to this connector

Examples:
The following example is the real life definition of the Mozilla Rhino attaching connector. 

 <extension
         point="org.eclipse.wst.jsdt.debug.core.connectors">
      <connector
            class="org.eclipse.wst.jsdt.debug.rhino.connect.RhinoAttachingConnector"
            id="rhino.attaching.connector">
      </connector>
   </extension>

API Information:
The class attribute is required and must implement org.eclipse.wst.jsdt.debug.core.jsdi.connect.Connector
